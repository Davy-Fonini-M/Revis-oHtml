História dos computadores

1° geração de computadores - Harvard Mark 1 (1944) Criado para contas complexas Colossus(1946) Criado para criptoanalises (Alemanha) Eniac(1946).

2° geração(1959 a 1965): esta geração os transistores foram usadas e eram mais baratos, consumiam menos energia, eram mais compactos no tamanho e também mais confiáveis e mais rápidos do que a da primeira geração as máquinas feitas de tubos de vácuo.

3° geração(1965 a 1970): Troca de valvulas para transistores - Circuito integrado transistor capacitores resistores

4° geração(a partir de 1971): Personal Computers Bill gates- Basics Apple - apple 1 > Macintosh

5° geração (futuro): Computadores quanticos, Criptomoedas, Metaversos e inteligencia artificial.

O que são Clients?

Lado Cliente→ Quando se consome o conteúdo/informação de um lugar da internet.

Hosperdar → Colocar o arquivo de internet no servidor, enviando e devolvendo a informação para o client.

Servidor → Retorna Arquivos Html, Css, Javascript isso é nomeado de (arquivos estaticos).

Cache → Quando envia a requisição para o servidor, o servidor retorna o conteúdo e salva na memoria cache do servidor deixando a usabilidade mais rápida.

Conceito Client → Diferentes resoluções, deve ser considerado na criação de websites.

O que são Servers?

Servidores → Ele não é acessivel apenas com internet, podendo ser local também. Nomeado> Computador central Quando há uma conexao entre esse computador central a rede é denomidada rede local. O servidor tende servir a muitos dispositivos fazendo requisição a ele. Existem diversos servidores, tipos Arquivos,segurança(firewall),streaming, E-mail e Web.

Entendendo a diferença entre listas ordenadas e não ordenadas

A principal diferença entre a lista ordenada e a lista não-ordenada está na maneira com que os itens de cada lista são apresentados - a lista ordenada enumera os elementos; a lista não-ordenada coloca marcadores antes de cada elemento.

Exemplo: 1- Esse é o item 1 (ol)

Exemplo: * Esse é o item 2 (ul)
